# pck3r commands

"install" command:

    $ pck3r install "somthing" :
    {
        nodejs,
        wine,
        ohmyzsh,
        flstudio,
        or ...
    }
    
"clear" command:

    $ pck3r clear
    {clear your terminal }

    $ pck3r sys upgrade
    (upgrade your oprating system)

    $ pck3r sys updgr
    (both, update and upgrade (full upgrade))


"tilix" command:


    $ pck3r tilix
    (tilix terminal ...)


"dotnet" command:

    $ pck3r install dotnet
    (installing .NET (dot net ) C0RE, ASP, MCS compiler , ...)    


"pkg" command:


    $ pck3r pkg <package name>"
    (search for packages ...)"


"update" command:


    $ pck3r update
    (update to last release from github.com/amzy-0/pck3r)



"version" command:


    $ pck3r version
    (this command show pck3r version)
